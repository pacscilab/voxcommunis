Background

Language family: Indo-European / Classical Indo-European / Indo-Iranian / Iranian / Central Iranian / Northwestern Iranian / Laki-Kurdish / Kurdish / Central Kurdish

  • Kurdish is spoken in a number of countries and has several dialects. Sorani (Central Kurdish) is one of these dialects and is spoken in several regions (Malmasi 2016, 89). It is mostly spoken in northern Iraq and western and northwestern Iran.
